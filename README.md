# 条件付き確率分布を用いた次元削減

このプロジェクトでは、条件付き確率分布 (Conditional Probability Distributions: CPDs) を用いた次元削減手法を実装しています。高次元データを低次元空間に写像しつつ、データ構造を保持することを目指しています。特に数値データに対応した前処理ステップ（ビニングやエンコーディングなど）を備えています。

## 特徴

- データ前処理:
  - 数値データの等頻度ビニング
  - 低頻度値の置換
  - カテゴリカル特徴量のエンコーディング
  - メモリ使用量の最適化
- 条件付き確率分布 (CPD) とその距離の計算
- ニューラルネットワークを用いた次元削減
- 早期終了 (Early Stopping)
- 低次元表現のエクスポート
